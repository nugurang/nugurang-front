# nugurang-front

This is a frontend repository of 'nugurang'.
You can go to nugurang-doc repository to see project description and installation manual.

## Installation

```bash
npm install
npm run init

```

## Run

```bash
npm run mock

```

```bash
npm run local

```

```bash
npm run dev

```
